City Coordinates Lookup API

Access per-country JSON files containing state and city data with coordinates.

API Usage

All JSON files are available at:

https://dhiyaaulauliyaa.github.io/city-coordinates-lookup/{country_id}_{country_code}.json

Available Countries

Example Usage

// Fetch US data
fetch('https://dhiyaaulauliyaa.github.io/city-coordinates-lookup/233_US.json')
  .then(response => response.json())
  .then(data => console.log(data));
